# Contributing

## Setup

Install Global Depenedencies:
```
npm install -g typings gulp
```

Clone the repository:
```
git clone https://github.com/Microsoft/PowerBI-Angular.git angular-powerbi
```

Install Dependencies:
```
npm install
typings install
``` 

Build:
```
gulp build
```
Or if using VScode: `Ctrl + Shift + B`

Test
```
npm test
```
